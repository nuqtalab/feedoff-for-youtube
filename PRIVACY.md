# Privacy Policy – FeedOff for YouTube

**Last updated:** 2026

FeedOff for YouTube is a Chrome extension developed by **NuqtaLab** to help users reduce distractions on YouTube by hiding feeds, Shorts, recommendations, autoplay, and other non-essential interface elements.

Your privacy is very important to us. This Privacy Policy explains how FeedOff for YouTube handles user data.

---

## 1. Data Collection

FeedOff for YouTube does **not collect, store, sell, or transfer any personal or sensitive user data**.

The extension does not collect:
- Personal information (name, email, address)
- Browsing history
- Video watch history
- Search queries
- Cookies
- Location data
- Financial or payment information

---

## 2. Local Storage

The extension uses Chrome’s `storage` API only to save user preferences locally, such as which YouTube sections are hidden or shown.

All settings:
- Are stored locally in the user’s browser
- Never leave the user’s device
- Are not shared with any third party

---

## 3. Permissions Usage

FeedOff for YouTube uses the following permissions:

### storage
Used to save user settings locally.

### activeTab
Used only when the user clicks a button in the extension popup to reload or reapply changes on the currently active YouTube tab.

### Host Permission
`https://www.youtube.com/*`  
Required to modify YouTube’s interface by hiding distracting elements. The extension does not operate on any other websites.

---

## 4. No Remote Code or Tracking

FeedOff for YouTube:
- Does not use remote code
- Does not load scripts from external servers
- Does not use analytics or tracking tools
- Does not display ads

All code runs locally in the browser.

---

## 5. Data Sharing

FeedOff for YouTube does not share, sell, or transfer user data to any third parties.

---

## 6. Children’s Privacy

FeedOff for YouTube does not knowingly collect data from children or adults. Since no personal data is collected, the extension is safe for all age groups.
